Rust bindings for Box3D

boxddd

Safe Rust APIs for Box3D 0.1.0, plus a native Bevy 0.19 teaching testbed for 3D bodies, colliders, joints, queries, events, debug draw, and interaction.

Examples On The Web

Every maintained example is visible in the gallery below. Native Bevy examples are represented with web previews and exact run commands; the current WASM entries are headless runtime proofs while live browser rendering is being promoted incrementally.
