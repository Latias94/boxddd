Rust bindings for Box3D

boxddd

Safe Rust APIs for Box3D 0.1.0, plus a native Bevy 0.19 teaching testbed for 3D bodies, colliders, joints, queries, events, debug draw, and interaction.

Repository Crates.io Docs.rs boxdd 2D sibling

Run The Native Testbed

The supported visual entry point is a native Bevy window. Browser live demos are future work while WASM provider packaging matures.

cargo run -p bevy_boxddd --features "debug-gizmos physics-picking" --example testbed_3d
